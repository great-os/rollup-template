# rollup-template
A template to build opensource library

# Suit for VSCode only

Tested vscode version:

- Version 1.33.1 (1.33.1)

# Steps to use this project

## Clone

```bash
git clone git@github.com:great-os/rollup-template.git
```

## Remove git history

```bash
rm -rf .git
```

## Init your own git

```bash
git init
```
